Sobre as vagas

Bolsista

O PET pode ter até 12 alunos bolsistas, que recebem uma bolsa de R$700,00 mensalmente.

A dedicação é de 20 horas semanais e a participação no PET garante ao aluno participação em 60 horas de atividades complementares elegíveis no curso de Ciência da Computação.

{% if VAGAS.BOLSISTA > 0 %} {{ VAGAS.BOLSISTA }} vagas {% else %} Sem vagas {% endif %}

Não-Bolsista

Caso todas as vagas para alunos bolsistas já estejam preenchidas, o PET pode ainda ter 6 alunos atuando como não-bolsistas.

As condições são as mesmas dos bolsistas, bem como o certificado, que também vale como Atividades Complementares.

{% if VAGAS.NAO_BOLSISTA > 0 %} {{ VAGAS.NAO_BOLSISTA }} vagas {% else %} Sem vagas {% endif %}

Colaborador

Caso as vagas para bolsistas e não-bolsistas já estiverem preenchidas, ainda é possível ser um membro do PET como colaborador.

Mesmo como colaborador você terá participação efetiva nos projetos, garantindo muito conhecimento e um grande incremento em sua graduação.

{% if VAGAS.COLABORADOR > 0 %} {{ VAGAS.COLABORADOR }} vagas {% else %} Sem vagas {% endif %}

Requisitos

Para se inscrever no processo seletivo, o candidato deverá:

  • 1.

    Ser aluno(a) regularmente matriculado(a) no curso de Bacharelado em Ciência da Computação da UFSCar, campus São Carlos;

  • 2.

    Caso o CANDIDATO já tenha tido vínculo com o PET BCC UFSCar (como bolsista ou não bolsista) e tenha sido desligado do programa, poderá ser inscrever novamente no processo seletivo se, e somente se, entre a data do seu desligamento e a inscrição tenha ocorrido a finalização de um semestre letivo com digitação de nota.
